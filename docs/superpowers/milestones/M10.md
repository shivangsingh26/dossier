**Goal:** Max tier. Daily scan of YourStory / Inc42 / TechCrunch / Crunchbase for new AI/ML startup funding rounds. Auto-checks if hiring. If yes → adds to user's watchlist. If not → drafts cold outreach to founder.

## Files (frontend)

- `frontend/app/(app)/market/page.tsx` (feed of new funded companies)
- `frontend/components/dossier/market/FundingCard.tsx` (company, round, founders, hiring status, action)
- `frontend/components/dossier/market/AddToWatchlistButton.tsx`

## Files (backend)

- `backend/src/dossier_api/routers/market.py` — `GET /market/feed`, `POST /pipeline/run agents=[market_intel]`
- Daily cron: 8am IST, fires `market_intel` for every Max user (5 credits each, auto-deducted)
- Cron implemented as scheduled task in `pipeline_worker.py` (no Celery needed)

## Acceptance criteria

- Max user opens `/market` → list of last 14 days' funded AI/ML startups
- Each card shows: company, round size, date, what they do (1 line), hiring? yes/no
- "Add to watchlist" → company added to user's `target_companies.json` for next watchlist run
- Daily 8am cron auto-runs for active Max users
- Founder cold-message drafts available for non-hiring startups (4 credits each)
- `pytest backend/tests/test_market.py -v` green
