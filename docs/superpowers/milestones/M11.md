# M11 — Payment Integration (Deferred)

> **Status:** Deferred. Do NOT start until paying-intent has been validated via the waitlist.

**Spec:** [docs/superpowers/specs/2026-05-18-dossier-saas-frontend-design.md](../specs/2026-05-18-dossier-saas-frontend-design.md) §12 (payment deferred).

**Goal:** Replace the "Upgrade → waitlist" stub with real Stripe checkout, subscriptions, and credit top-ups.

## Prerequisites before starting M11

- ≥ 20 active users have signed the waitlist for Pro
- ≥ 5 users have signed the waitlist for Max
- At least 3 dogfood weeks completed with stable MVP
- Stripe India account approved (Stripe India + Razorpay are both options — decide at M11 start)

## Files (frontend)

- `frontend/app/(app)/billing/page.tsx` (current subscription + history)
- `frontend/components/dossier/billing/CheckoutButton.tsx` (Stripe.js)
- `frontend/components/dossier/billing/CreditTopupModal.tsx`

## Files (backend)

- `backend/src/dossier_api/routers/billing.py`
  - `POST /billing/checkout-session` (creates Stripe session)
  - `POST /webhooks/stripe` (subscription created/updated/deleted, invoice paid)
  - `POST /billing/topup-credits`
- `backend/src/dossier_api/services/stripe_service.py`
- New `subscriptions` table in `accounts.db`

## Acceptance criteria

- User clicks "Upgrade to Pro" → Stripe checkout → payment → webhook fires → tier flips to Pro, credits set to 500
- Monthly subscription renewal via webhook resets credits
- Credit top-up: ₹X buys N extra credits (rates TBD when M11 starts)
- Cancel subscription → tier reverts to Lite on next renewal
- Full Stripe test suite with `stripe-mock`

## Pricing (intent only — set at M11 start)

- Pro: ₹999/month
- Max: ₹2,999/month
- Credit top-ups: ₹500 / 200 credits (rough — finalize before launch)
- Annual: 2 months free (price × 10)

## What this milestone does NOT touch

- Invoicing, GST, tax compliance (use Stripe's hosted invoices initially; consult CA if scaling)
- Refunds, chargebacks — manual via Stripe dashboard initially
- Multi-currency — INR only at launch, USD added after
